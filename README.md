# 🎨 高级个人作品集网站

一个融合 Apple 极简美学与 Linear 深色主题的专业个人网页。

## ✨ 设计亮点

### 🎯 设计灵感
- **Apple 风格**: 摄影棚级的产品展示、SF Pro 字体美学、极简主义设计
- **Linear 风格**: 深色主题、精致的毛玻璃效果、优雅的紫色点缀色

### 🎨 视觉特色
- **深色主题**: #010102 极深背景，营造高端专业感
- **渐变点缀**: Linear 标志性的薰衣草蓝 (#5e6ad2) 作为主色调
- **精致排版**: Space Grotesk + Inter 字体组合，完美字间距
- **毛玻璃导航**: 20px 模糊效果的固定导航栏
- **噪点纹理**: SVG 噪点叠加，增加质感深度

### 🎭 动画效果
- **页面加载**: 依次淡入上浮的入场动画
- **滚动进度**: 顶部渐变进度条
- **自定义光标**: 点+环组合，hover 时放大
- **卡片悬停**: 上浮 + 边框高光 + 顶部渐变线
- **时间线节点**: hover 时发光效果
- **技能标签**: hover 时变色上浮

## ✏️ 自定义修改

### 修改颜色
编辑 CSS 变量：
```css
:root {
    --primary: #5e6ad2;  /* 主色调 */
    --canvas: #010102;   /* 背景色 */
    --ink: #f7f8f8;      /* 文字颜色 */
}
```

### 修改字体
替换 Google Fonts 链接和字体变量：
```css
:root {
    --font-display: 'Your Display Font', sans-serif;
    --font-body: 'Your Body Font', sans-serif;
}
```

### 修改内容
直接编辑 HTML 中的文本内容，保持结构不变。

## 🎯 设计原则

1. **极简主义**: 少即是多，每个元素都有存在的意义
2. **视觉层次**: 通过大小、颜色、间距建立清晰的信息架构
3. **微交互**: 恰到好处的动画，增强用户体验
4. **一致性**: 统一的设计语言贯穿整个页面
5. **性能优先**: 纯 CSS/JS 实现，无外部依赖
